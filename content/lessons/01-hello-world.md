# 第1章：Hello, World!

欢迎来到 Python 的世界！这一章我们来写第一个 Python 程序。

## 什么是 Python？

Python 是一种**简单易学**的编程语言。它被广泛用于网站开发、数据分析、人工智能等领域。Python 的语法接近自然语言，非常适合编程入门。

## 第一个程序

在编程世界里，第一节课永远是从 `Hello, World!` 开始的。让我们来看看：

```python
print("Hello, World!")
```

> 💡 `print()` 是 Python 的**打印函数**，用于在屏幕上输出文字。

## print 还能做什么？

`print()` 可以输出数字、计算结果，还可以一次输出多个内容：

```python
print(42)
print(3 + 5)
print("我的年龄是", 18)
```

## 注释

在 Python 中，`#` 号后面的文字是**注释**，不会被运行。注释用来给代码写说明：

```python
# 这是一条注释，不会被执行
print("这行代码会运行")
# print("这一行被注释掉了，不会运行")
```

> 💡 注释是给**人**看的，不是给电脑看的。写注释是个好习惯！
