# 第10章：元组和集合

除了列表和字典，Python 还有两种重要的数据类型：元组和集合。

## 元组 tuple

元组和列表类似，但**创建后不能修改**（不可变）：

```python
# 创建元组
point = (3, 5)
colors = ("红", "绿", "蓝")
single = (1,)    # 只有一个元素时，逗号不能省略

print(point[0])    # 3（支持索引）
print(point[-1])   # 5（支持负索引）

# 元组不能修改
# point[0] = 10   # ❌ 报错！
```

### 为什么用元组？

```python
# 1. 作为函数返回值
def get_position():
    return (10, 20)  # 返回多个值

x, y = get_position()  # 解包
print(x, y)  # 10 20

# 2. 作为字典的键（列表不行）
locations = {
    (35, 139): "东京",
    (40, 116): "北京",
}

# 3. 比列表更安全（数据不会被意外修改）
```

## 集合 set

集合是无序的、**不重复**的元素集合：

```python
# 创建集合
fruits = {"苹果", "香蕉", "苹果", "橙子"}
print(fruits)  # {"苹果", "香蕉", "橙子"}（重复自动去重）

# 空集合
empty_set = set()  # 注意：{} 是空字典
```

### 集合操作

```python
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

# 并集
print(a | b)  # {1, 2, 3, 4, 5, 6, 7, 8}

# 交集
print(a & b)  # {4, 5}

# 差集
print(a - b)  # {1, 2, 3}

# 对称差集（不同时在两个集合中的元素）
print(a ^ b)  # {1, 2, 3, 6, 7, 8}
```

### 集合的常用方法

```python
nums = {1, 2, 3}

nums.add(4)          # 添加
nums.remove(2)       # 删除（不存在会报错）
nums.discard(10)     # 安全删除（不存在不报错）
print(len(nums))     # 长度
print(3 in nums)     # 成员检查

# 列表去重
numbers = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(numbers))
print(unique)  # [1, 2, 3, 4]
```

## 三种数据结构的对比

| 特性 | 列表 list | 元组 tuple | 集合 set |
|------|-----------|-----------|---------|
| 符号 | `[]` | `()` | `{}` |
| 可变 | ✅ | ❌ | ✅ |
| 有序 | ✅ | ✅ | ❌ |
| 可重复 | ✅ | ✅ | ❌ |
| 用途 | 存储序列 | 不可变数据 | 去重/集合运算 |
